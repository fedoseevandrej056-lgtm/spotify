# Spotify Music Player - Flutter iOS
## Приложение для проигрывания музыки в стиле Spotify

Полнофункциональное приложение для проигрывания музыки для iOS, разработанное на Flutter с дизайном, вдохновленным Spotify.

### ✨ Особенности

- **Темная тема в стиле Spotify** - минималистичный дизайн с фирменным зеленым цветом (#1DB954)
- **Музыкальный плеер** - полная поддержка воспроизведения MP3, M4A, WAV, AAC
- **Локальная библиотека музыки** - загрузка песен из файлов
- **Мини-плеер** - компактный плеер внизу экрана
- **Полноэкранный плеер** - просмотр информации о песне с прогрессией воспроизведения
- **Контроль воспроизведения** - play/pause, next/previous, shuffle, repeat
- **Управление плейлистами** - создание и управление плейлистами
- **Реактивный интерфейс** - использование Provider для управления состоянием

### 🚀 Начало работы

#### Требования

- Flutter 3.0.0 или выше
- Dart 3.0.0 или выше
- Xcode 14 или выше (для iOS)



```bash

```

### 🎵 Как использовать

1. **Загрузка песен**
   - На главном экране нажмите кнопку "Add" или "Import Songs"
   - Выберите MP3, M4A, WAV или AAC файлы из памяти устройства
   - Песни буду добавлены в вашу библиотеку

2. **Воспроизведение**
   - Нажмите на песню в списке для начала воспроизведения
   - Используйте мини-плеер внизу экрана или откройте полноэкранный плеер

3. **Управление**
   - Play/Pause - средняя большая кнопка
   - Previous/Next - боковые кнопки со стрелками
   - Shuffle - включить перемешивание
   - Repeat - переключание режима повтора

### 🎨 Обзор компонентов

#### AudioPlayerService
Управляет воспроизведением аудио, позицией в треке и состоянием плеера.

```dart
final playerService = Provider.of<AudioPlayerService>(context);
await playerService.playSong(index);
```

#### MusicLibraryService
Управляет загрузкой песен из файловой системы и плейлистами.

```dart
final libraryService = Provider.of<MusicLibraryService>(context);
await libraryService.importSongsFromFiles();
```

#### HomeScreen
Отображает список всех песен в библиотеке с возможностью выбора и воспроизведения.

#### NowPlayingScreen
Полноэкранный плеер с информацией о текущей песне, прогрессом воспроизведения и элементами управления.

#### PlayerControls
Набор кнопок управления плеером (play, pause, next, previous, shuffle, repeat).

### ⚙️ Конфигурация iOS

Прав доступ настроены в `ios/Runner/Info.plist`:

- **NSLocalNetworkUsageDescription** - доступ к локальной сети
- **NSAppleMusicUsageDescription** - доступ к музыкальной библиотеке
- **UIBackgroundModes** - аудио в фоне
- **UIFileSharingEnabled** - обмен файлами через iTunes

### 🔧 Решение проблем

**Приложение не видит музыку**
- Убедитесь, что песни находятся в папке Documents приложения
- Формат файлов: MP3, M4A, WAV, AAC

**Звук не воспроизводится**
- Проверьте громкость устройства
- Убедитесь, что в iPhone не активирован режим "Не беспокоить"
- Перезагрузите приложение

**Проблемы при сборке**
- Запустите `flutter clean` и затем `flutter pub get`
- Обновите Pod зависимости: `cd ios && pod install && cd ..`
